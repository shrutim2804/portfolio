# My Personal Portfolio

This is my personal portfolio website built with React to showcase my projects, skills, and experience as a Computer Science student.

## 🚀 Live Demo
View it live here: [https://shrutimishra-portfolio.vercel.app/](https://shrutimishra-portfolio.vercel.app/)

## 🛠️ Tech Stack
- **Frontend:** React, HTML, CSS, JavaScript
- **Deployment:** Vercel

## 📌 Features
- Clean, modern UI design
- Fully responsive across devices
- Shows my projects, skills, internships, and achievements
- Integrated social links (GitHub, LinkedIn, Email)

## 📦 How to Run Locally
1. Clone this repository: `git clone https://github.com/shrutim2804/portfolio.git`
2. Navigate to the folder: `cd portfolio`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
5. Open `http://localhost:3000` in your browser
