# Shruti Mishra — Personal Portfolio

A personal portfolio website built with **React**, designed to present my projects, technical skills, internships, and achievements as a Computer Science Engineering student.

🔗 **Live Site:** [shrutimishra-portfolio.vercel.app](https://shrutimishra-portfolio.vercel.app/)

| Layer | Technologies |
|-------|-------------|
| Frontend | React, HTML5, CSS3, JavaScript |
| Deployment | Vercel |

## Features

- **Modern UI** — Clean, minimal design with smooth navigation
- **Fully Responsive** — Optimized for desktop, tablet, and mobile
- **Project Showcase** — Highlights key academic and personal projects
- **Skills & Experience** — Covers technical skills, internships, and achievements
- **Social Integration** — Quick access to GitHub, LinkedIn, and Email

---

## Getting Started

Follow these steps to run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/shrutim2804/portfolio.git

# 2. Navigate into the project directory
cd portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.
